O ficheiro a ser executado é o "generator.py".

No entanto, para mudar certos parâmetros, como a ordem, o ficheiro, ou o alfa, isso é mudado
no fim do ficheiro "fcm.py".
Para mudar o número de caracteres a ser gerado, é preciso alterar a variavel "generate_size" que se encontra no ficheiro generator.py

Caso esteja em Linux, é necessário ter o numby instalado.
Se não tiver, só é preciso fazer no terminal:

$ sudo pip install numpy scipy



O ficheiro "texto.txt" contem mais de 850 mil caracteres.
