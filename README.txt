

Os textos a serem computados para a formação do modelo encontram-se na diretoria Idioms.
Os textos a serem adivinhados encontram-se na diretoria guess_texts.
A diretoria Classifier contém 10 ficheiros para cada língua para efetuar a classificação da precisão do sistema

O programa permite escolher entre introduzir manualmente um texto, ler um dos textos predefinidos da diretoria guess_texts,
    ou então introduzir o nome de um ficheiro customizado, dando o caminho absoluto ou então se estiver na pasta do projeto, apenas o nome do ficheiro
